  • Optimization for Fault Tolerant Neural Network Architecture (Project Member)

    • To deploy deep learning algorithms on resourcelimited scenarios, an emerging device-resistive random access memory (ReRAM) has been regarded as promising via analog computing. To investigate a better fault tolerant neural network architecture, we systematically explored the weight drifting tolerance of different neural network components and used Bayesian optimization to search for the optimal neural architecture robust to weight drifting. The search space included dropout rate, Gauss or Laplace noises distributions. The experiments demonstrated that our algorithmic framework had outperformed the state-of-the-art methods on various tasks, such as image classification and object detection. This work will be contributed to Nature Electronics in the future

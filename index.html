Recent Projects

John Hopcroft Center for Computer Science, SJTU, Research Intern

  • Optimization for Fault Tolerant Neural Network Architecture (Project Member)

    • Although the emerging device-resistive random access memory (ReRAM) has been regarded as promising to deploy deep learning algorithms on resourcelimited scenarios, its practicability is primarily limited due to the weight drifting of neural networks caused by multi-factor reasons. To investigate a better fault tolerant neural network architecture, we systematically explored the weight drifting tolerance of different neural network components and used Bayesian optimization to search for the optimal neural architecture robust to weight drifting. This work will be contributed to Nature Electronics in the future

  • Deep High-Dynamic Range (HDR) Image Compression (Project Leader)

    • Recently, deep learning has been widely explored and proven to be a promising approach in standard dynamic range (SDR) image compression. However, few learning-based methods have been proposed on high dynamic range (HDR) image compression, and existing traditional methods suffer from compression artifacts or limited improvements. Therefore, we proposed a novel deep learning-based method for end-to-end HDR image compression based on deep neural networks. This work will be submitted to ACM Multimedia (MM) 2021.

    Robotics Institute, School of Mechanical Engineering, SJTU, Master

  • A Brain-Actuated Robotic Arm System Using Non-invasive Hybrid Brain-Computer Interface and Shared Control Strategy (Project Leader)

    • To improve the performance of non-invasive brain-actuated robotic arm system, the shared control strategies are tried as an alternative solution. However, most of the existing shared control methods set the arbitration rules manually, which highly depends on the specific tasks and developer’s experience. Hence, we proposed a novel shared control model that automatically optimized the control commands in a dynamical way based on the context in real-time control. Besides, we employed the hybrid brain-computer interface (BCI) to better allocate BCI resources. This work has been published in Journal of Neural Engineering.

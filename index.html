John Hopcroft Center for Computer Science, SJTU, Research Intern

  • OoDHDR-codec: Out-of-Distribution Generalization for HDR Image Compression (Project Leader)

    • Jan. 2021 - Sept. 2021

    Due to the wide luminance distribution of high dynamic range (HDR) images and the lack of large standard datasets, developing a deep model for HDR image compression is much more challenging. To tackle this issue, we view HDR data as distributional shifts of SDR data and the HDR image compression can be modeled as an out-of-distribution generalization (OoD) problem. Specifically, we proposed a novel OoD HDR image compression framework, which improved the compression model's OoD generalization ability for HDR image compression.

  • Optimization for Fault Tolerant Neural Network Architecture (Project Member)

    • May. 2021 – Jul.2021

    To deploy deep learning algorithms on resource-limited platforms, analog deep neural networks represent apromising solution. However, the practicability of analog deep neural networks has thus far been limited by weight drifting. To investigate a better fault tolerant neural network architecture, we systematically explored the weight drifting tolerance of different neural network components and used Bayesian optimization to search for the optimal neural architecture robust to weight drifting.

    Robotics Institute, School of Mechanical Engineering, SJTU, Master

  • A Brain-Actuated Robotic Arm System Using Non-invasive Hybrid Brain-Computer Interface and Shared Control Strategy (Project Leader)

    		•

    Sep. 2019 - Nov. 2020

    To improve the performance of non-invasive brain-actuated robotic arm system, the shared control strategies are tried as an alternative solution. However, most of the existing shared control methods set the arbitration rules manually, which highly depends on the specific tasks and developer’s experience. Hence, we proposed a novel shared control model that automatically optimized the control commands in a dynamical way based on the context in real-time control. Besides, we employed the hybrid brain-computer interface (BCI) to better allocate BCI resources.

  • A Shared Control Strategy for Reach and Grasp of Multiple Objects Using Robot Vision and Non-invasive Brain-Computer Interface (Project Member)

    		•

    Nov. 2018 - Apr. 2019

    It is ambitious to develop a brain-controlled robotic arm to perform activities of daily living using brain–computer interfaces (BCIs). Despite much progress achieved, this mission is still very challenging mainly due to the poor decoding performance of BCIs. A shared control strategy is developed in this work to realize flexible robotic arm control for reach and grasp of multiple objects. With the assistance provided by robot vision, the subject could manipulate robotic arm using a simple motor imagery-based BCI with binary output.
